诊断功能通过任务与审核中心提供。包含：

  • 摄入任务状态与重试
  • 审核队列处理
  • 操作日志查看
  • 知识库健康检查
  • 诊断包导出（已自动脱敏）

🔒 隐私说明

  • 所有数据存储在本地 SQLite 数据库和配置文件中。
  • API Key 单独加密存储，不进日志、不进诊断包。
  • 诊断包自动脱敏，不包含密钥、原文内容或个人信息。
  • SignalVault 不上传任何数据到云端。
  • LLM API 调用只传输分析所需的文本内容。

⚠ 免责声明

SignalVault 不是投资建议工具。

本软件仅帮助用户结构化整理公开投资信息。所有分析结果基于 AI 模型对公开内容的提取，不代表投资建议。投资决策请自行判断，风险自担。

  • yt-dlp — YouTube 字幕提取
  • Playwright — 浏览器自动化测试
