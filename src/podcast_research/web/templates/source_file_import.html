导入流程

1. 上传

选择本地文本文件上传。系统自动识别文件类型和编码。

2. 预览

系统提取文本内容，判断入库资格，检测重复，推荐归档方式。

3. 确认

你决定是否归档。确认后写入 SourceArchive，文件名自动 sanitize。

当前支持

文件类型 说明
.md Markdown 文档，提取首个 H1 作为标题，保留正文
.txt 纯文本文件，自动检测编码（UTF-8 / GB18030）
.html / .htm HTML 页面，提取标题和正文，清除脚本和样式

本轮不支持 .pdf / .docx / .xlsx / 图片 / 音视频 / 压缩包。这些格式将在后续版本支持。
