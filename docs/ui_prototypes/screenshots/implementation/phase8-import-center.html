youtube Web 已支持

YouTube 视频

从频道候选池或视频字幕进入分析

适合公开投资长视频、播客字幕。系统会提取观点、标的、风险、信号和原文时间戳。

  1. 添加或刷新频道
  2. 筛选新视频
  3. 生成投资报告
管理 YouTube 已有频道会显示新视频候选池。
zsxq Web 已接入

知识星球

只读导入已订阅星球主题

适合导入指定 group/topic 的长帖、讨论和附件元数据。不会发布、评论、点赞或修改星球内容。

  1. 检查登录状态
  2. 选择 group/topic
  3. 导入或进入分析
导入知识星球 读取依赖本机 zsxq-cli 登录状态。
web Web 已支持

网页链接

文章、播客笔记、研究页面

粘贴 URL 后先解析预览，系统检测重复并推荐归档、关联精读笔记或跳过。

  1. 粘贴 URL
  2. 查看预览和冲突
  3. 确认写入
粘贴网页 普通网页默认归档为资料，不自动生成投资报告。
tracked Web 已支持

固定信息源

反复更新的外部网页源

适合 All-In 中文笔记等固定来源。刷新后发现新条目，支持批量预览和确认导入。

  1. 添加跟踪源
  2. 刷新发现新条目
  3. 批量确认导入
管理固定源 不支持的 URL 可改用单网页导入。
file 文本 Web 支持，PDF 后端支持

文件导入

文本文件、HTML、PDF 研报

Web 已支持 .md/.txt/.html/.htm 预览归档；PDF 已具备预览、提取、分析和页码证据能力。

  1. 上传或选择文件
  2. 预览提取质量
  3. 归档或进入分析
上传文本文件 PDF Web 原生上传分析待接入；当前保留 CLI/PDF 专用工作流。

安全边界

导入中心不会自动写入或自动外发。

先预览所有资料先进入预检/预览。
再确认用户确认后才写入 Vault 或 DB。
保留证据YouTube 时间戳、PDF 页码、ZSXQ topic 引用。
不做投资建议只整理观点和信号，不输出买卖建议。
